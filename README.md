# Pyghtcast
An api wrapper around Lightcast API. Provided as is.

Borrows heavily from [EmsiApiPy](https://github.com/calebjcourtney/EmsiApiPy/tree/master).
## Installation

```
pip install git+https://github.com/Dallas-College-LMIC/pyghtcast
```
